# Linear Regression From Scratch

## What this project does
Built Linear Regression algorithm from scratch using Python without sklearn.

## How it works
- Calculates slope (m) and intercept (b) manually
- Uses least squares formula
- Trained on placement dataset

## Tools Used
- Python
- NumPy
- Pandas
- Matplotlib

## Key Finding
RMSE is 0.83 LPA and have a 27% of average error

## Key Learning
Built the math behind Linear Regression manually instead of 
using sklearn's built-in model.
